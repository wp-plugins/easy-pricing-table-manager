Easy Pricing Table plugin allow you to create Beautiful Pricing table or Comparision Table with smooth hover effects in just a few minutes.

== Description ==

Description: Easy Pricing Table plugin allow you to create Beautiful Pricing table or Comparison Table with smooth hover effects in just a few minutes. You can Embed in any post/page using shortcode <code>[easy-pricing-table table_name="Pricing Table Name"]</code> or you can add through widgets.

= Features =
* Easy to use
* Fully Responsive
* Nice Hover Effects
* Simple and light weight
* Clean and Nice Backend User interface
* You can Display Table using short-codes or by useing Widget
* Can create as many tables needed and seperate settings for eaach


= Demo =

[Click here](http://jwthemes.com/easy-pricing-table/) to see Easy Pricing Table Manager in live action.

== Installation ==

1. Upload the `Easy Pricing Table` folder to the `/wp-content/plugins/` directory
1. Activate Easy Pricing Table Plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Easy Pricing Table` tab that appears in your admin menu
1. Add to any page using short-code `[easy-pricing-table table_name="Pricing Table Name"]` or you can use widget too.
